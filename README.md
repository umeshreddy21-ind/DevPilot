# 🚀 DevPilot

A modern Flask-based Project Management System that helps users manage projects and notes with authentication, analytics, file uploads, PDF export, and a responsive interface.

---

## 🌐 Live Demo

🔗 https://YOUR-RENDER-LINK.onrender.com

---

## 📌 Features

- ✅ User Registration & Login
- ✅ Secure Password Hashing
- ✅ Dashboard Analytics
- ✅ Project Management (CRUD)
- ✅ Notes Management (CRUD)
- ✅ Search Projects & Notes
- ✅ Filters & Sorting
- ✅ Pagination
- ✅ File Uploads
- ✅ PDF Export
- ✅ Responsive Bootstrap UI
- ✅ Dark Mode
- ✅ PostgreSQL Database
- ✅ Custom 404 & 500 Pages

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/DevPilot.git
```

Go to the project

```bash
cd DevPilot
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
